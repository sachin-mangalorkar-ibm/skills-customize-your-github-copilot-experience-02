# 📘 Assignment: Algorithmic Problem Solving with Lists and Dictionaries

## 🎯 Objective

Practice building step-by-step solutions using Python lists and dictionaries. You will solve common algorithmic challenges such as counting frequencies, finding duplicates, and selecting top values.

## 📝 Tasks

### 🛠️ Build a Frequency Counter

#### Description
Write a function that counts how many times each value appears in a list of numbers.

#### Requirements
Completed program should:

- Implement `count_frequencies(numbers)` and return a dictionary.
- Use each number as a key and the number of occurrences as the value.
- Work correctly for an empty list.
- Print the result for the sample input provided in the starter code.

### 🛠️ Find Duplicates in a List

#### Description
Write a function that returns all duplicated values from a list, without repeating duplicates in the result.

#### Requirements
Completed program should:

- Implement `find_duplicates(numbers)` and return a list of duplicates.
- Include each duplicated value only once in the result.
- Return duplicates in sorted order.
- Return an empty list if there are no duplicates.

### 🛠️ Select Top Students by Score

#### Description
Use dictionary data to identify students with scores at or above a threshold.

#### Requirements
Completed program should:

- Implement `top_students(scores, threshold)` where `scores` is a dictionary.
- Return a list of student names whose score is greater than or equal to the threshold.
- Return names in alphabetical order.
- Print the result for the sample data and threshold in the starter code.
